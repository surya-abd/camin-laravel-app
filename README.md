# Final Project Oprec MCI 2022
  
## Nuno.Label
Aplikasi ini merupakan e-commerce untuk brand nuno.label

## Fitur Aplikasi
List fitur pada aplikasi ini adalah sebagai berikut.
1. Melihat list produk
2. Menambah produk
3. Mengedit detail produk
4. Menghapus produk
